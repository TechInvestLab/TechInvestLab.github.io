Archived version installation instructions
Last updated: 28 feb 2014 :: 1K

Released version works in MS Windows (XP, Vista, 7) only, no other OS versions available.

Download .rar archive. No installation is required; just extract archive contents into a folder on your hard drive.

Avoid putting the program or data on virtual drives, external flash drives or network drives, as system may have restrictions to access them. Always test program's ability to access folders you have selected for your data.

If you are updating from a previous version - choose separate folder or backup extensions, pattern definitions and data sample files if you've changed them.

If you are installing to a new folder - copy files dot15926.cfg and input.log from the folder with your old executable to the folder with the new executable. This will preserve your configuration of data sources and their properties, and keep the history of your console queries and algorithms. Copy \editor\extensions folder if you are changing extensions.

To execute the Editor please run \editor\dot15926.exe.

To uninstall – just delete folders.
